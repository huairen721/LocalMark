## 简介

一个专门用于给身份证等敏感证件加水印的工具，完全基于浏览器本地API，无任何网络请求，欢迎👏大家 star 、 issue 、 pr。

## 功能

- 批量添加水印
- 完全本地，不依赖网络
- 0 依赖
- 批量下载水印图片
- 支持设置水印颜色、间隔、透明度、字体大小

## 说明

- 在原版的基础上，使用`Material 3 Expressive设计语言`美化显示内容。
- 根据图片数量，自动调整预览图片的大小。

0 依赖, 拎包入住

## 使用

- 直接双击下载好的 `index.html` 文件，用浏览器打开即可使用，无需安装。
- 点击左侧的「选择图片」区域，从电脑里挑选要加水印的照片，一次可以选多张。
- 在「水印文字」框里填写你想加的内容，比如"仅供XX平台使用"。
- 在「样式调整」里可以改颜色、透明度、大小和间距，按个人喜好调就行。
- 右侧会实时显示加好水印的效果，满意后点击「下载全部」即可保存到电脑。
